# gravispy
Gravitational lensing simulations written in Python.

### Requirements

+ numpy
+ vispy
+ sympy
+ scipy

### Installation

To install in developer mode, go to the base directory that you've cloned and execute the pip command:

```bash
pip install --user -e .
```
